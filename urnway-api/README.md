# Urnway API

Backend for Urnway, an AI-native travel super app built on Mezo.

## Stack

- Express
- TypeScript
- Drizzle ORM
- Postgres (Supabase)
- Redis
- viem
- Railway

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run db:generate`
- `npm run db:migrate`
- `npm run db:push`
- `npm run db:studio`

See `.env.example`.
